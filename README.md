# FDA-FOIA-2026-6007 — Document Navigator

A searchable, filterable index of the documents released in **FDA FOIA request
FDA-FOIA-2026-6007** (Bioresearch Monitoring / BIMO inspection records for
COVID-19 vaccine clinical trials — Pfizer-BioNTech & Moderna; CBER Partial
Response 1).

## What this is

`index.html` loads `documents.json` (7,417 documents) and renders a table where
each row links to the document:

- **New** documents (not in the PHMPT public corpus) are hosted here in `files/`.
- **PHMPT** documents link to their individual page on phmpt.org, with the source
  ZIP as a backup.
- Where available, a **Drive** link opens the file on the
  [coviddocuments.com](https://coviddocuments.com) Google-Drive navigators
  (Pfizer BLA / Pfizer EUA / Moderna BLA).

## Files

- `index.html` — the navigator UI (vanilla JS, no build step)
- `documents.json` — the document index
- `files/` — 330 newly-released documents hosted for direct download
- `.nojekyll` — tells GitHub Pages to serve files verbatim

## Note

One newly-released file, *125742_S1_M1_4.3 Literature References.pdf* (137 MB),
exceeds GitHub's 100 MB per-file limit and is not hosted here; its row links to
the original MuckRock bundle instead.

Regenerated by `build_site.py` + `host_new_files.py` in the parent project.
